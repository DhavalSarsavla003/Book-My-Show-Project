# Book My Show

A movie ticket booking application that allows users to browse movies, select seats, and book tickets online.

## Features

- **Movie Listings**: Browse available movies with details like release date and price
- **Seat Selection**: Interactive seat selection with real-time status updates
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **User-Friendly Interface**: Clean and intuitive UI for easy navigation

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Styling**: CSS

## Installation

### Prerequisites
- Node.js installed
- MongoDB running locally or cloud connection
